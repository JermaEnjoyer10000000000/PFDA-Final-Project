## Description
1-2 sentence description of what it will do and how it relevant to media and digital arts.
The video download tool will enable users to paste and download video URLs in various video formats to store on their PC.
The problem this solves is that it allows users to watch any video they choose whenever they desire, directly from their PC in any format and quality.


## Features
- Feature 1: Recognizing acceptable URL
	- Through using try and except to identify acceptable URL
- Feature 2: Converting URL to desired video format
	- Can be done using similar code to aurora lab assignment
- Feature 3: Providing support for different video formats
	- Accomplished through incorporating MoviePy into program
- Feature 4: Providing different video resolutions
  - Capturing video frames and upscaling or downscaling them using MoviePy before storing to PC

## Challenges
- Converting URL to desired video format 
- Ensuring the associated video URL remains the same during download
- Optimizing the download process

## Outcomes
Ideal Outcome:
- Several options for video formats and qualities and fully optimized download process

Minimal Viable Outcome:
- URL is capable of being downloaded to PC via one video format and quality
